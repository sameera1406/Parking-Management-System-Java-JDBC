# Parking Management System (Java + JDBC + MySQL)

## Description
A console-based Parking Management System built using Java and JDBC.
It allows users to:
- Add users
- Add parking slots
- Book slots
- View users, slots, and bookings

## Technologies Used
- Java
- JDBC
- MySQL
- VS Code

## How to Run
1. Create database `ParkingManagement`
2. Update DB credentials in code
3. Compile and run `ParkingManagementApp.java`
